AdvocaIA — Site Profissional de Advocacia Inteligente

## Desenvolvimento (React)

- Requisitos: Node.js LTS (`node`, `npm`)
- Dependências: `npm install`
- Servidor local: `npm run dev`
- Build de produção (pasta `dist/`): `npm run build`

Imagens do site ficam em `public/img/` (por exemplo `escritorio.png` e `advogados.png`).

📘 Sobre o Projeto

O AdvocaIA é um projeto fictício desenvolvido para representar uma advocacia moderna, automatizada e digitalmente inteligente.
Mais do que um simples site, ele foi criado como um exercício avançado de engenharia de prompt e integração de Inteligência Artificial aplicada ao desenvolvimento web.

Todo o conteúdo — texto, estrutura e identidade visual — foi gerado com suporte de IA, guiado por prompts detalhados e estratégicos, projetados para alcançar resultados coerentes, funcionais e visualmente consistentes.

🎯 Objetivo

Explorar o potencial da IA Generativa como ferramenta criativa e técnica, aprimorando minha habilidade de comunicar, instruir e direcionar inteligências artificiais para entregas de alta qualidade e valor visual.

⚙️ Stack e Ferramentas Utilizadas

HTML5, CSS3, JavaScript — base do front-end

IA Generativa (GPT / Claude / Gemini) — para criação textual e concepção visual

Engenharia de Prompt — estruturação estratégica de instruções para IA

Automação de Conteúdo — otimização de geração de texto e identidade do site

Figma (conceito visual) — prototipagem e alinhamento estético

🧩 Competências Demonstradas

Engenharia de Prompt (comunicação técnica e persuasiva)

Desenvolvimento Front-End Responsivo

Criação de conteúdo com IA Generativa

UX Writing e Arquitetura da Informação

Automação de fluxos criativos e prototipagem assistida por IA

💡 Insight

O AdvocaIA simboliza o futuro da criação digital — onde a criatividade humana e a inteligência artificial trabalham lado a lado para desenvolver soluções únicas, eficientes e escaláveis.
